1. Syntax highlighting

This plugin provides a math prettifying plugin for snm.

To use, put this in your syntax.yaml:

   package: snmmath-0.0.1 (If this document is not up to date, you may need to find the correct version number). 
   module: Text.Syntax.Math 
   symbol: math

See the snm user guide on how to use this plugin.

A list of all symbols processed is given below.

The left column is the input text, the right column is the output symbol.

All output symbols will be given the css class math_symbol.
